## Radianite v0.1.6 — Dashboard Polish & Smarter Settings

Radianite is a lightweight Windows companion app for VALORANT players and streamers. It shows your live game activity on Discord, adds a rank overlay to OBS without API keys, and keeps everything running locally on your PC — no Riot password required.

This release improves dashboard readability, makes status timestamps easier to understand, and moves more settings and queue handling into the Rust backend for a more reliable app experience.

### ✨ Features

- Display your live VALORANT activity on Discord.
- Add a live rank overlay to OBS without API keys or extra accounts.
- See your current rank, Rank Rating, and recent RR change.
- Automatically detect whether VALORANT is open, in menus, matchmaking, in a match, or watching a replay.
- View live match details in a clean desktop dashboard.
- Show localized VALORANT agent, map, and rank artwork in the dashboard.
- Minimize Radianite to the Windows system tray.
- Optionally launch Radianite when Windows starts.
- Check for and install updates from the app.
- View release notes directly from the updates card.
- Runs locally without requiring your Riot password.
- Simple Windows installation and lightweight performance.

### 🆕 What’s New in v0.1.6

- **Relative timestamps** — Recent sync and update-check times now use clear labels such as “just now” and “2 minutes ago.”
- **Accurate app uptime** — The status bar now measures uptime from when Radianite starts.
- **Better live match details** — Long match values can be expanded instead of being permanently cut off.
- **Improved window controls** — Title-bar buttons now provide clearer hover and interaction feedback.
- **Smoother update checks** — Update activity and recent-check timing are more stable and easier to understand.
- **More reliable settings** — Settings startup and queue-name normalization now run in Rust, reducing frontend work and keeping behavior consistent.
- **General maintenance** — Updated dependencies, documentation, and app metadata for the v0.1.6 public beta release.
