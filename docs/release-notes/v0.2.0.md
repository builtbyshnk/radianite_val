## Radianite v0.2.0 — Svelte Migration & Custom Overlays

Radianite is a lightweight Windows companion app for VALORANT players and streamers. It shows your live game activity on Discord, adds a rank overlay to OBS without API keys, and keeps everything running locally on your PC — no Riot password required.

This release rebuilds the desktop interface with Svelte 5, introduces customizable overlay themes and privacy controls, and fixes Discord preview behavior when your VALORANT session changes.

### ✨ Features

- Display your live VALORANT activity on Discord.
- Add a live rank overlay to OBS without API keys or extra accounts.
- See your current rank, Rank Rating, and recent RR change.
- Automatically detect whether VALORANT is open, in menus, matchmaking, in a match, or watching a replay.
- View live match details in a clean desktop dashboard.
- Show localized VALORANT agent, map, and rank artwork in the dashboard.
- Customize the stream overlay and hide sensitive player details.
- Minimize Radianite to the Windows system tray.
- Optionally launch Radianite when Windows starts.
- Check for and install updates from the app.
- View release notes directly from the updates card.
- Runs locally without requiring your Riot password.
- Simple Windows installation and lightweight performance.

### 🆕 What’s New in v0.2.0

- **Rebuilt with Svelte 5** — The desktop interface has moved from React to Svelte 5 while preserving the existing Radianite experience.
- **Custom overlay themes** — Choose a visual theme for the OBS overlay from Radianite’s settings.
- **Privacy controls** — Hide detailed player information from the overlay when you want a cleaner or more private stream layout.
- **Improved overlay layout** — The hidden-details header is now centered for a more balanced presentation.
- **Better Discord preview behavior** — Preview activity now clears when a VALORANT session ends, and its uptime timestamp is more accurate.
- **Updated Discord link** — The Rich Presence link now points to the Radianite project website.
- **Stronger Windows validation** — Added UI parity tests, lifecycle coverage, visual baselines, and a Windows performance benchmark for the frontend migration.
- **Improved diagnostics** — Riot connection diagnostics now update through app events, making status changes more responsive and easier to troubleshoot.
- **Developer tooling** — Added consistent frontend formatting, linting, and updated GitHub Actions configuration.
