## Radianite v1.0.0 — A More Polished Desktop Companion

Radianite is a lightweight Windows companion app for VALORANT players and streamers. It shows your live game activity on Discord, adds a rank overlay to OBS without API keys, and keeps everything running locally on your PC — no Riot password required.

This release improves how Radianite starts and runs in the background, makes the desktop interface more responsive, and refines the overlay, map, and settings experience.

### ✨ Features

- Display your live VALORANT activity on Discord.
- Add a live rank overlay to OBS without API keys or extra accounts.
- See your current rank, Rank Rating, and recent RR change.
- Automatically detect whether VALORANT is open, in menus, matchmaking, in a match, or watching a replay.
- View live match details in a clean desktop dashboard.
- Show localized VALORANT agent, map, and rank artwork in the dashboard.
- Customize the stream overlay and hide sensitive player details.
- Minimize Radianite to the Windows system tray.
- Optionally launch Radianite when Windows starts.
- Check for and install updates from the app.
- View release notes directly from the updates card.
- Runs locally without requiring your Riot password.
- Simple Windows installation and lightweight performance.

### 🆕 What’s New in v1.0.0

- **Low-resource background launch** — Radianite can now start in the background with a lighter startup path, helping it stay out of the way when Windows launches.
- **Responsive settings and title bar** — Settings dialogs and window controls now adapt better to smaller window sizes.
- **Clearer overlay actions** — Unavailable overlay actions now provide clearer guidance.
- **Improved map readability** — Live map labels are easier to read at a glance.
- **More reliable locale validation** — Locales are validated as part of the build process, helping keep translated UI text consistent.
- **Faster development and release builds** — Incremental development builds and release configuration have been refined for a smoother build experience.
- **Updated branding and project links** — Repository URLs and app branding have been refreshed.
