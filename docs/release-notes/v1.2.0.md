## Radianite v1.2.0 — Appearance, Presence & Overlay Refinements

Radianite is a lightweight Windows companion app for VALORANT players and streamers. It shows your live game activity on Discord, adds a rank overlay to OBS without API keys, and keeps everything running locally on your PC — no Riot password required.

This release adds appearance and startup preferences, improves Discord Rich Presence across more VALORANT states, and refines the OBS overlay and desktop controls.

### ✨ Features

- Display your live VALORANT activity on Discord.
- Add a live rank overlay to OBS without API keys or extra accounts.
- See your current rank, Rank Rating, and recent RR change.
- Automatically detect whether VALORANT is open, in menus, matchmaking, in a match, or watching a replay.
- View live match details in a clean desktop dashboard.
- Show localized VALORANT agent, map, and rank artwork in the dashboard.
- Customize the stream overlay and hide sensitive player details.
- Minimize Radianite to the Windows system tray.
- Optionally launch Radianite when Windows starts.
- Check for and install updates from the app.
- View release notes directly from the updates card.
- Runs locally without requiring your Riot password.
- Simple Windows installation and lightweight performance.

### 🆕 What’s New in v1.2.0

- **Appearance settings** — Choose your preferred app appearance; the selection and window state are now remembered between launches.
- **Start minimized** — When Radianite launches automatically with Windows, it can now start minimized.
- **Richer Discord activity** — Discord Rich Presence now better distinguishes Premier, Range, matchmaking, idle, and solo matchmaking states, with improved localized text and artwork.
- **More accurate rank lookups** — Rank data is now tied to the current season, preventing stale season results.
- **Refined OBS overlay preview** — Overlay cards, theme selection, preview sizing, and positioning have been adjusted for a cleaner configuration experience.
- **Improved desktop controls** — Title-bar controls now include tooltips, and tooltip surfaces have been restyled for better clarity.
- **Updated startup presentation** — The initial boot theme has been refreshed for a more polished first impression.
- **Maintenance updates** — Frontend and Rust dependencies, pre-commit checks, and CI configuration have been updated.
